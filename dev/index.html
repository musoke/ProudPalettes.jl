ProudPalettes

ProudPalettes is a Julia package providing color palettes derived from pride flags.

API

  • ProudPalettes.ProudPalettes
  • ProudPalettes.PROUD_PALETTES
ProudPalettes.ProudPalettes — Module

Colour palettes

For flags composed of horizontal stripes, the colors in each array are ordered from bottom to top. This results in color bars with an intuitive order.
